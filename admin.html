Configurer le token GitHub (une seule fois)
Comment obtenir un token GitHub :
  1. Va sur github.com/settings/personal-access-tokens/new
  2. Token name : securite-ECRAN (ou ce que tu veux)
  3. Expiration : 1 an (ou "No expiration" si tu veux)
  4. Repository access : Only select repositories → sélectionne securite-ECRAN
  5. Permissions → Repository permissions → Contents : Read and write
  6. Clique sur Generate token et copie le token (commence par github_pat_)
  7. Colle-le ici et enregistre
Le token reste sur ton navigateur (localStorage). Il n'est pas envoyé ailleurs qu'à GitHub.
